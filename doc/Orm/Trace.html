Class Trace

Description
Description | Vars (details) | Methods (details)

Allow display stacktrace and other informations in a console or on the page

example

  1.           Trace::debug('i am here !');
  2.           Trace::info('process finished');
  3.           Trace::warn('Be carfull : too much results will be displayed');
  4.           Trace::error('oups houston, we have got a problem... :( ');

the property $level of this class defines the level of the messages to displaying So a level of "WARN" will only display warning and error messages By default the level is setted to WARN
